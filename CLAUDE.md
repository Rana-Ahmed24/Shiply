# Shiply Egypt Project Rules

## Product
A premium marketplace platform connecting travelers with customers who want products brought from abroad into Egypt.

## Main Marketplace Roles
- Travelers
- Customers
- Admins

## Tech Stack
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Stripe
- Framer Motion

## Design Style
- Premium fintech + travel startup aesthetic
- Inspired by Airbnb, Stripe, Uber, and StockX
- Mobile-first
- Clean spacing
- Soft shadows
- Rounded 2xl corners
- Minimal clutter
- Dark navy  + teal accents
- use the images in the refrences folder for a refrence



## Coding Rules
- Use TypeScript strict mode
- Use reusable components only
- Avoid duplicate code
- Avoid inline styles
- Use Tailwind utility classes
- Use modular folder structure
- Use server components where possible
- Keep files clean and scalable

## UX Rules
- Prioritize trust
- Keep onboarding simple
- Make the marketplace easy to understand
- Mobile-first responsive design

## Core Features
- Traveler listings
- Customer requests
- Messaging system
- Escrow/deposit system
- Verification system
- Reviews and ratings
- Admin moderation
- Search and filtering
